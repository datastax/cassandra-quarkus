An Apache Cassandra(R) extension for Quarkus.

== Getting the extension

The extension artifact is published in Maven central, under the group id
link:http://search.maven.org/#search%7Cga%7C1%7Cg%3A%22com.datastax.oss%22.quarkus%22[com.datastax.oss.quarkus]:

[source,xml]
----
<dependency>
  <groupId>com.datastax.oss.quarkus</groupId>
  <artifactId>cassandra-quarkus-client</artifactId>
  <version>${cassandra-quarkus.version}</version>
</dependency>
----

== Compatibility

The extension is compatible with Quarkus version `3.31.0` and higher.

Quarkus link:https://quarkus.io/blog/java-17/[requires Java17 or greater] since Quarkus 3.7 so this project does as well.

Quarkus 3.x requires at least Maven version `3.8.2` or higher to build correctly (see link:https://github.com/quarkusio/quarkus/wiki/Migration-Guide-3.0#maven-versions[Quarkus3 notes]).
